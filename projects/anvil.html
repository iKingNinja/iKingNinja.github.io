Anvil


Anvil is an application made in C# using the .NET Framework. Anvil will make customizing your Roblox Client and Studio a lot less annoying as it will automatically change files for you when Roblox updates. You just have to select the files that you want Anvil to replace Roblox’s with.

Anvil was developed following a scheme to be as fast as possible and to complete all the tasks before the game launches. Anvil runs parallel threads which means that all the tasks will be executed simultaneously.

Anvil lets you customize the cursor, the death sound and even class images for Roblox Studio so that if you want to use Vanilla you don’t have to manually change the file every time Roblox Studio updates.
